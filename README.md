# PrimeForm

PrimeForm is a Flutter-based fitness application focused on personalized training plans,
progress tracking, and practical AI-assisted coaching.

---

## Tech Stack
- Flutter (stable)
- Dart
- Android
- Firebase
- Isar (local database)

---

## Getting Started

### Install dependencies
```bash
flutter pub get

### Run In Debug Mode
```bash
flutter run
```
### Build Release APK
```bash
flutter build apk --release
```
### Build Release App Bundle
```bash
flutter build appbundle --release
```
